### To install node packages:

```bash
 npm i
     or
 yarn install
```    
### To start the server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## App description

This is a Employee Management application to manage the employee in an Organization.
